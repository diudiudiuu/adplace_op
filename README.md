# 广告投放管理面板

## 启动客户端

    npm run tauri dev  

## 打包客户端

    npm run tauri build

## node 版本

    22.12.0

## tauri 升级

    cargo clean
    cargo install tauri-cli --force
    cargo update
    cargo build


## rust 升级

    rustc更新
    rustup update stable

    mac更新
    brew update
    brew upgrade rust
